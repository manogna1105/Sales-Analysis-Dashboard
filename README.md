# Sales-Analysis-Dashboard
## Project Overview
This project is an interactive data analytics dashboard designed to visualize sales performance, identify trends, and provide actionable insights for business growth. It processes raw sales data to track Key Performance Indicators (KPIs) such as Total Sales, Profit Margins, and Regional Performance.

### Objective:  
To answer critical business questions regarding revenue trends, best-selling products, and customer segmentation.

### Key Features
*  nKPI Tracking: Real-time view of Total Revenue, Total Profit, Order Count, and Average Order Value.

* Interactive Filtering: Dynamic filters allowing users to sort data by Year, Region, Product Category, and Customer Segment.

* Trend Analysis: Line charts visualizing month-over-month (MoM) and year-over-year (YoY) sales growth.

* Geospatial Visualization: Interactive map showing sales distribution across different states and cities.

* Category Performance: Breakdown of top-performing product categories and sub-categories.

### Technologies Used
* nProgramming Language: Python

* Data Processing: Pandas, NumPy

* Visualization: Matplotlib, Seaborn, Plotly

* Dashboard Framework: Streamlit (or Power BI / Tableau / Dash)

* Data Source: CSV / SQL / Excel

### Project Workflow
* Data Cleaning: Removed duplicates, handled missing values, and standardized date formats.

* Data Transformation: Created new metrics like "Profit Margin %" and "Sales per Customer."

* Exploratory Data Analysis (EDA): Identified correlations and seasonal patterns.

* Dashboard Development: Built the visual interface for end-users.

### Key Insights Derived
* Seasonality: Sales consistently peak during Q4 (November-December), driven by holiday demand.

*  Regional Leaders: The West Region accounts for the highest percentage of total profit.

* Product Focus: "Technology" products have the highest profit margin, while "Furniture" has the highest shipping costs.
